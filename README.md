# Developer's Guide

```
Node.js
Sails.js
Postgresql
Docker
```

## Install dependencies

* [Docker for Mac](https://store.docker.com/editions/community/docker-ce-desktop-mac)

## Add .env file

```
$ cd sails-docker-compose
$ touch dev.env prod.env
```

## Start development server

```
$ cd sails-docker-compose
$ make dev
# open localhost:1337
```
